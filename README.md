## Valentine Website (React + Express)

Proyek ini adalah website spesial Hari Valentine untuk pasangan kamu.

### Struktur

- `server/` – Backend Express, serve API dan file statis.
- `client/` – Frontend React (via CDN) berisi halaman romantis.

### Cara Menjalankan

1. Masuk ke folder `server`:
   - `cd server`
2. Jalankan server:
   - `node index.js`
3. Buka browser ke:
   - `http://localhost:3000`

Pastikan kamu sudah install Node.js dan npm di komputer kamu.
